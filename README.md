# 📖 Overview

**C Functions & Recursion** is a collection of C programs that demonstrate the power of **user-defined functions**, **parameter passing**, **call by value**, **call by reference**, and **recursive programming**.

The programs are designed to improve code modularity, reusability, and logical thinking while introducing recursion as a technique for solving computational problems.
